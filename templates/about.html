About This Project

Welcome to the About page for the Automated Retinal Imaging project. This tool is developed to demonstrate the application of machine learning in analyzing retinal fundus images for potential signs of Diabetic Retinopathy (DR).

Project Goal

The primary goal of this project is to provide an interactive example of how convolutional neural networks (CNNs), specifically a DenseNet121 architecture, can be trained and used for image classification tasks in a medical context. It aims to:

  • Illustrate the process of image analysis using a pre-trained model.
  • Showcase the integration of a machine learning model within a web application using Flask.
  • Demonstrate the use of explainability techniques like Grad-CAM to visualize which parts of an image the model focuses on.
  • Provide general information about Diabetic Retinopathy and its associated risks (cardiac, glaucoma) based on the model's output.

Technology Used

This project utilizes the following key technologies:

  • Flask: A micro web framework for Python, used for the backend server and handling web requests.
  • TensorFlow/Keras: An open-source machine learning library used for building and running the deep learning model (DenseNet121).
  • OpenCV (cv2): Used for image processing tasks, including preparing images for the model and generating Grad-CAM heatmaps.
  • NumPy: A fundamental package for scientific computing in Python, used for numerical operations on image data.
  • HTML, CSS, JavaScript: For the frontend web interface, handling user interaction and displaying results.
  • Google Generative AI (Gemini API): Used to provide explanatory summaries and information related to the predicted diagnoses.

How It Works

When you upload a retinal image, the Flask backend receives the image and passes it to the loaded DenseNet121 model. The model processes the image and outputs a prediction indicating the severity level of Diabetic Retinopathy (from 'No_DR' to 'Proliferate_DR'). The application then uses Grad-CAM to generate a heatmap highlighting the areas of the image that were most influential in the model's prediction. Finally, relevant information and an explanation are provided based on the predicted diagnosis.

Important Disclaimer

This tool is built for **educational and demonstrational purposes only**. It is designed to show how machine learning can be applied to medical imaging, but it is **not a medical device** and should **not** be used for actual medical diagnosis or treatment decisions.

Machine learning models, including the one used here, can make mistakes. The predictions and information provided by this tool may be inaccurate. **Always consult with a qualified healthcare professional** for any health concerns, diagnosis, or treatment. Do not rely on the results of this tool for medical advice.

The information provided regarding cardiac and glaucoma risks is general and based on known associations with Diabetic Retinopathy stages. It is not a substitute for a comprehensive medical evaluation.
